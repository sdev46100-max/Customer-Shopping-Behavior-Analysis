# Customer-Shopping-Behavior-Analysis
This project analyzes customer shopping behavior using MySQL. The goal of the analysis is to understand purchasing patterns, revenue distribution, and product performance based on customer demographics.

# Project Overview
The dataset was explored using SQL queries to identify key insights related to customer purchases, product ratings, and revenue trends across different customer segments.

# Analysis Performed
Calculated total revenue generated by male and female customers.

Analyzed revenue contribution by different age groups.

Identified the top 5 products with the highest review ratings.

Retrieved the top 3 most purchased products within each category.

Compared average purchase amount across different customer groups.

# SQL Concepts Used
Aggregation functions (SUM, AVG, COUNT)

GROUP BY and HAVING clauses

Conditional Logic (CASE Statements)

Window functions and ranking for product analysis
